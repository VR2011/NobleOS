An operating system inside a single website.

## Features
**NobleOS has many things included in the operating system, with more to come in the future. Some of these features are:**

- An about page
- A simple calculator
- A browser
- A start menu + app selector
- A clock
- A terminal
- A simple file system
- Wallpaper picker
- A theming menu that has 5 different themes to choose from!
    - **The following themes in the OS are...**
    - Regular (Gold-ish theme)
    - Blue
    - Pink
    - Dark
    - Light
